Exercice 3: test sur une proportion

nbsucess <- sum(data$taille>180)
n <- nrow(data)
(p <- nbsucess/n)
## [1] 0.56
p*n>10 & (1-p)*n>10
## [1] TRUE
alpha <- 0.1
Binf <- p-sqrt(p*(1-p)/n)*qnorm(1-alpha/2)
Bsup <- p+sqrt(p*(1-p)/n)*qnorm(1-alpha/2)
cat("Intervalle de confiance: [",Binf," ; ",Bsup,"]\n")
## Intervalle de confiance: [ 0.4783516  ;  0.6416484 ]
p0 <- 0.5
tcalc <- (p-p0)/sqrt((p0*(1-p0)/n))
pvalue <- 2*(1-pnorm(abs(tcalc)))
cat("p-valeur:",pvalue,"\n")
## p-valeur: 0.2301393

\(p_0\)=0.5 est dans lintervalle de confiance de niveau 90%, la p-valeur est grande, il nexiste donc pas de risque raisonable permettant de rejeter lhypothse H0.
